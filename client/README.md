# Project descriptions

- Git-Conneced
  A social media networking app for early career developers. I help maintain the project through code refactoring and review, feature planning, research and development. It uses 3rd party Oauth to connect to Discord, Twitter and GitHub to provide a central location to follow members of the community.
- Infinite-Stats
  An application to check your career stats for the game Halo Infinite. Users get detailed information of their overall performance totals, indevidual matches and across various playlists. This project integrates Autocode's Halo Infinite API to keep player stats up date with their most recently played match.
- Jobbot-the-Bot
  This script is designed to reduce wasted time in the job search for developers. It scrapes remote web development opportunites from indeed, cleans and shortens the URL via my Shrinkenator API, stores the data to a database and integrates with discord providing a bot that periodically posts new opportunities with a short description and link to my channel. It can also respond to commands to post jobs scraped on a spicifc date.
- Nade Spartan
  An Open Broadcaster Software browser source overlay game. Inspired by the popular twitch overlay drop game, and a channel point redemption I felt dissatisfied with. When a new game is started the target spartan spawns at a random X position along the bottom of your OBS browser source. On typed command a nade element is positioned at a random X position off screen above and assigned a random X and Y velocity. Using requestAnimationFrame a Draw and Update function create a smooth falling effect that calculates a score based on distance from center within a danger zone of the target spartans position. Nades that land in the dangerzone have their twitch username added to a high score scoreboard.
- Todo List
  A simple todo list, every developer has to have one. It utilizes JWT token authentication and MaterialUI for design.
- Shrinkenator
  A simple full stack URL shortening service with an ironically long name.
